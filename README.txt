# PRODIGY_DS_01 – Population Distribution Visualization

## Task Overview

This project was completed as part of my Data Science Internship at Prodigy InfoTech.

The objective of this task is to visualize population distribution data using graphical representations. The project focuses on understanding demographic patterns through data visualization techniques.

## Dataset

The dataset contains demographic information including:

* Age
* Gender

## Technologies Used

* Python
* Pandas
* Matplotlib

## Visualizations Created

### 1. Age Distribution Histogram

A histogram was created to visualize the distribution of individuals across different age groups and identify population trends.

### 2. Gender Distribution Bar Chart

A bar chart was created to compare the frequency of different gender categories within the dataset.

## Project Workflow

1. Load the dataset using Pandas.
2. Explore and understand the dataset structure.
3. Analyze age and gender distributions.
4. Create a Histogram for age distribution.
5. Create a Bar Chart for gender distribution.
6. Save and display the visualizations.

## Output Files

* **age_distribution.png**
* **gender_distribution.png**

## Learning Outcomes

* Data visualization using Python.
* Understanding demographic data analysis.
* Creating Histograms and Bar Charts using Matplotlib.
* Extracting meaningful insights from population data.
